## Crypto — public chains / L1 / L2

- **再论公链 (public-chain V4, 2025)** — Main contradiction is **资产发行效率**, not decentralization; finance lens (Crypto ≠ Web3), asset-penetration North Star, "1995 internet" moment. Signature 协议层 vs 应用层 split (different metric weights) ⇒ protocol layer = settled ETH oligopoly, application layer open. SOL won via C-end asset-issuance + org efficiency; L1-L2 are competitors (post-4844 ETH captures ~8% of OP-Rollup value, "tax-sacrificing marketing"); "Ethereum ≠ ETH" creation-without-capture. Banded targets ETH $3k/$6k/$10k, SOL $250/$500.
- **公链现状及发展趋势初探(上) (2022)** — First-principles: 公链 = the internet's value protocol layer; 去中心化是手段, 安全是目的, gas = security cost. CAP → 不可能三角 → multi-chain inevitable + scaling is the main contradiction. ETH (61% mcap) is the only "立住" consensus layer; modular blockchain (Rollup + Danksharding) most likely to break the trilemma.
- **公链现状及发展趋势初探(下) (2022)** — 不可能三角 as the evaluation spine; 场景→需求→规模→基础设施 loop; 0-1/1-10/10-100 + 飞轮 (Solana case). Five-dimension chain rubric (叙事/运营/生态/资本/团队); ETH dual valuation (gold + Visa) with 3-scenario probability-weighted targets; ZK wins long-term, OP leads now.
- **以太坊的进化之路 (2024)** — Re-orders the trilemma: 去中心化是目的 >> 可扩展性 >> 安全. ETH = decentralized cloud-computing infrastructure; slow iteration is *deliberate* (链下治理 + 模糊共识). Core-EIP vs ERC split; 搏流量靠资产→保留存靠叙事→真用户靠应用; next app innovation likely on neither ETH nor today's L2s; endgame = near-non-iterating ToB infra whose latent moat is architectural decentralization.
- **基于Layer2的以太坊生态研究 (2024)** — Period thesis = **L1×L2 COMPLEMENTARY** (协议层×应用层互补; L2 拉用户与资产, ETH 仍是核心货币) — *not* a competitive/tax-collapse frame. Metaphors: **L2 = 以太坊外接硬盘 / 商业模式=云计算 / 像应用发行商 / Killer App 自建应用链**. Top-of-funnel: Crypto=资产≠Web3(流量); 资产先于用户. 合规两大胜负手 (牌照/商业模式/基因三层). Four-factor L2 eval **生态>运营>资本团队>技术**; head-to-head **OP > Arbitrum** (OP Stack B2B2C, EF正统, Paradigm/a16z); Op-Rollup mainstream next 12mo vs ZK (Cairo门槛/流动性差/V神意志). Empirical anchor: tx/active-addr converged to L2 **but 资产流量/TVL 未明显迁移 L1**. Bottom-up ETH revenue = L1 gas + L2 存储费; PS 熊150/牛45; **ETH next-cycle ~$6000-8000**.

## Crypto — exchanges / stablecoins / DeFi / PerpDex

- **Coinbase公司分析 (V2, 2022)** — Web3 = next-gen internet (Read→Write→Own); North Star = MTU市占率 (3%); high-ARPU two-sided flywheel; compliance moat = double-edged sword; **revenue decomposition (price-appreciation vs price-neutral)** proves ~80% BTC-driven; **BTC-CAGR×WACC grid** shows CB stock can't beat holding BTC ⇒ must find decorrelated revenue.
- **Coinbase公司更新 (2023)** — Crypto = financial asset (not currency/commodity); maps Coinbase onto TradFi institutions ("一站式服务浏览器", ≈ internet brokerage, PS 3-5x); three core indicators MTU/TVL/VOLUME; cyclical non-linear 3-scenario valuation (4-10x); Uniswap fee-switch upside.
- **从合规的角度看Coinbase (V5, 2023)** — Asset-precedes-user reframe (缺新资产, not 缺用户); 交易所两大核心 = 资产流量 + 合规; 混业→分业经营 inevitable (托管/清结算 spun off first); 稳定币 = gateway/Killer App (~100x runway); 3-scenario valuation (合规护城河 weighted 50%).
- **美国现货分发渠道分析 (Coinbase vs Robinhood, 2025)** — 融合 (TradFi×blockchain) lens; the three US crypto bills *remove* the compliance barrier ⇒ Coinbase's real moat (regulatory ambiguity) erodes; **ETF = final winner** (~50% of US spot); fees converge; 资沉 growth = market β not mindshare; derivatives = subsidy-driven wash; both could be >$300B in 5-10yr; North Star = 入金用户数 + AUM.
- **从传统金融交易基建的角度理解Hyperliquid (2025)** — 收单/执行/清结算 settlement-stack; Crypto commercializes the clearing layer ⇒ revolution against TradFi; 流量×流动性 multiplier (HL = liquidity, no traffic); organic-vs-wash unit economics (OI/Vol 1.12, Pearson 0.63 "不是一波人", vampire-attack counterfactual); dual PS/PE valuation with 监管折价 → fair ~$48.7B (priced-in); regulation = black swan.
- **Hyperliquid研究修订版 (2025)** — Treat as 链上交易所 not a chain; PE-style (99% buyback); dual-track (historical PS vs comp), 监管折价, probability-weighted; share-of-Binance forecasting; conclusion: 公允价格, market已 price-in.
- **Coinbase不仅仅是交易所 (V6, 2024)** — **BULLISH SuperApp / undervalued** call (NOT "护城河融化"). 合规护城河权重 ~50%; "2024合规元年"; moat = **合规牌照→地域事实性垄断 + 美国用户垄断 (双垄断)**; two engines = 行业Beta + 地区事实性垄断. 合规势能递减 **交易所>稳定币>RWA**; 怎么做合规 = 运营+基因+商业模式+用户心智 (非合规→合规几乎不可能). Crypto 第一性=点对点公共账本; 三次记账范式; 三大资产属性; 五个创新原则. Playbook: 现货=资产(多快新稳, 公链/Launchpad/Venture); 合约=渠道+代理(KOL 首单返佣70%); 国际站=存量变现. 银行收入: 散户55%/机构4%, 稳定币19%/质押10%/托管2%/利息5%. Forward P&L: 24Q1>$15亿, peak季度~$46亿, 市值 **>$1000亿 (≥5x)**.
- **从Marathon Digital看比特币算力行业 (2024)** — **BULLISH-with-a-number** (NOT "long-term underperforms BTC"). Centerpiece = **杠杆比率回归** (MARA bull 6x→1.5x this cycle, CLSK 3x, MSTR 0.6x, bear ~1x) ⇒ BTC $150K → MARA ~3x+, full upside 8-10x from $16K low; **股价弹性比BTC更大**. Peer set = **MARA/CLSK/MSTR** (BTC的法币映射; 一级折价 vs 二级); relative pick **CLSK** (lower cost, smaller). **P/B primary** (MARA 4.84/CLSK 4.09); warn off mNAV/AI-HPC/REIT (anachronistic for 2024). **4-stage 减半周期** (Rising Bull/Goldrush/Inventory Flush/Shake Out) + hashrate lags price 60-120天. Second curve = **交易手续费 + 铭文/Ordinals** (not AI). Contrarian temper: 财富效应递减 (312%→134%→18%). Anchors: MARA 24.7 EH/s, 15,126 BTC held, cost $29,547/$17,417, D/E 0.2, 48 employees, 第三方托管. α inputs: CEO 背景, 内部持股, Core Scientific 破产对照.

- **从AI的角度来看Meta (2025)** — "一流团队对冲二流生意"; 北极星 ranking 流量规模>新媒体形式>转化率; ad-revenue decomposition spine; 有数无端 (IDFA reverse-proof); Core AI (β) vs Gen AI (α); 开源即护城河; TikTok dual-path non-threat; 3-scenario PE valuation (~60% upside).

- **AI 模型的计算与存储 (2026)** — GPT-2-small worked example; "two numbers decide everything": 计算量 (FLOPs/MFU) + 存储量 (VRAM). matrix mul 2MNK; block 24d²+4sd; step ≈6NT; MFU ~10% busts peak illusion; training 16 bytes/param = 8× inference; activation ∝ s².
- **AI 算力与模型基础 (2026)** — Y = M·X compute lens; 三透镜 显存/算力/通信; Roofline + arithmetic intensity (decode = HBM-bound, prefill/training = FLOPS-bound); software stack as a margin lever (~2×/gen); 长程 ≠ 长上下文; "Token 即电力" as a discussion; 瓶颈→受益方→护城河→利润 速查表.
- **大模型与自动驾驶 (2023)** — 两大拐杖 (激光雷达 + 高精地图) are wrong crutches; 数据-成本恶性循环; 局部端到端 viable / 全局端到端 infeasible; 世界模型 = GPT-training analogy; if corner cases unsolved, production cars stay L2+ long-term; pure-vision = best cost-performance (Tesla, FSD >25% penetration / 32亿km).
- **英伟达：拿着锤子找钉子的并行计算信仰者 (2024)** — **PASSIVE valence**: NVIDIA = 大算力场景**发现**公司, 只能造更好的锤子但**不能创造下游场景**, 收入被下游产业被动决定 (拿着锤子找钉子 — NOT an active 收税官 empowerment thesis). Scenario-picking = **全行业试错, 非独到视野** (场景随机突现; survives via scientist ecosystem + 蹭热点). North Star = **next-gen 10x model OR 100M-DAU killer app** (Killer App 70% / 10x model 30%; hard to appear ⇒ ceiling). Demand sizing: 训练~3.5M H100($120B), 推理~17M A100($260B) off cloud-capex×50%. Valuation: SOTP (Graphic PS + Compute&Network PE/G, CAGR 42%, PE/G 1.2); $3T fair-in-wave, $1.5T "AI stops". Timing = market **already over-discovered/over-valued**.
- **从AIGC角度看微软 (2023)** — North Star = **"微软能否再造新操作系统 / 重构流量入口"** (weight **70% AI端重构 + 30% AI生态**); 重造OS ⇒ ~5x, 失败 ⇒ max ~3x. **大模型竞争四维度 场景>>数据>>算法>>算力** ⇒ **"微软暂时领先, 但被Google超越可能性大"** (Google赢场景+数据, 微软仅算法+算力). 大模型=下一代计算机(不是互联网/不解决连接); 爆发标志=Windows-95级模型+算力下放消费级显卡. Anchors: ROE中枢~10%, 权益乘数2-3x, 20yr rev CAGR~9%→65万亿日元, 股息3-5%. Verdict: Mitsui **defensive cash-cow hold BUT current buy timing poor (USD/JPY FX risk = co-equal #1 risk)**.
